## 👗주제
vgg19를 활용한 의류 HS Code 자동 분류 시스템

## 🐶개요
텍스트 데이터가 아닌, 의류 이미지 데이터를 딥러닝을 활용하여 자동으로 HS Code 분류가 가능한 시스템을 제안한다. 이를 통해 품목분류의 정확성을 높이고 통관 절차를 간소화할 수 있다.
